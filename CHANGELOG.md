# Changelog

## [0.4.10] - 2026-07-09

### Changed
- `작업 컨트롤` 바를 단순화했습니다. 별도 상태 드롭다운과 `동시 적용`/`담당자 적용` 버튼을 없애고, 에이전트 드롭다운에서 담당자를 고른 뒤 `완료 처리`/`대기 전환`을 누르면 상태 변경과 담당자 할당이 함께 적용되도록 바꿨습니다. `미할당만 적용` 버튼은 그대로 유지됩니다.
- 확장 프로그램 버전을 `0.4.10`으로 올렸습니다.

## [0.4.9] - 2026-07-09

### Added
- 상단 `작업 컨트롤` 바에 상태 드롭다운(대기/진행중/완료/변경 안함)을 추가하고, 상태와 담당자를 함께 골라 한 번에 반영하는 `동시 적용` 버튼을 추가했습니다.

### Fixed
- 상태와 담당자를 동시에 변경할 때 담당자 지정이 조용히 무시되던 문제를 고쳤습니다. 이제 두 값을 한 요청에 함께 담아 시도합니다.
- 드래그 선택 중 포인터를 빠르게 움직이면 중간 행이 선택에서 누락되던 문제를 고쳤습니다. 이동 이벤트를 보간(interpolate)해 지나친 모든 행을 빠짐없이 처리합니다.

### Changed
- 확장 프로그램 버전을 `0.4.9`로 올렸습니다.

## [0.4.8] - 2026-07-09

### Changed
- 작업별 상태 표시를 `완료` 체크박스에서 `대기/진행중/완료` 3단계 상태 드롭다운으로 바꿨습니다. 실제 Freshservice 작업 status(1/2/3)를 그대로 반영/변경합니다.
- 확장 프로그램 버전을 `0.4.8`로 올렸습니다.

## [0.4.7] - 2026-07-09

### Changed
- 개별 작업의 담당자 드롭다운을 본인으로 기본 선택해두던 동작을 되돌렸습니다. 대신 `작업 컨트롤`(일괄 처리) 바의 담당자 드롭다운에서 본인 계정을 목록 맨 위에 두고 기본 선택되도록 했습니다. 이미 다른 담당자를 선택한 상태라면 그 선택은 유지됩니다.
- 확장 프로그램 버전을 `0.4.7`로 올렸습니다.

## [0.4.6] - 2026-07-09

### Fixed
- 확장 프로그램을 업데이트한 뒤 이미 열려 있던 티켓 화면에서 발생하던 `Extension context invalidated` 원문 오류 대신, 페이지 새로고침을 안내하는 메시지를 표시하도록 했습니다. 감지되면 불필요한 폴링/감시도 중단해 반복 실패를 막았습니다.
- 확장 프로그램 버전을 `0.4.6`으로 올렸습니다.

## [0.4.5] - 2026-07-09

### Added
- 작업 row를 드래그 없이 클릭만 해도 선택/해제되도록 해서, 단일 작업 선택 시 드래그 없이 바로 처리할 수 있게 했습니다.
- 아직 담당자가 지정되지 않은 작업의 개별 작업 컨트롤 바 담당자 드롭다운에 본인 계정이 기본 선택되도록 했습니다. (화면 표시만 반영되며 실제 할당은 저장하지 않습니다)

### Changed
- 완료된 작업의 `완료` 라벨을 색이 채워진 배지로 바꾸고, 완료된 작업 row에 초록색 강조 표시를 추가해 시인성을 높였습니다.
- 확장 프로그램 버전을 `0.4.5`로 올렸습니다.

### Removed
- 클릭 즉시 체크되던 기존 작업별 `선택` 체크박스를 제거해, 드래그·단일 클릭 선택 방식으로 작동 방식을 통일했습니다.

## [0.4.4] - 2026-07-09

### Added
- 작업 row를 드래그해서 여러 작업을 한 번에 선택하거나 선택 해제할 수 있게 했습니다.

### Changed
- 확장 프로그램 버전을 `0.4.4`로 올렸습니다.

## [0.4.3] - 2026-07-09

### Added
- 작업 일괄 컨트롤 바에 `대기만 선택`, `미할당 선택`, `선택 비우기` 액션을 추가했습니다.
- 선택된 작업 수와 선택 대상의 대기/완료/미할당 현황을 일괄 컨트롤 바에서 바로 확인할 수 있게 했습니다.

### Changed
- 작업 row 선택 상태를 `선택됨` 라벨과 강조 스타일로 더 명확하게 표시하도록 개선했습니다.
- 일괄 컨트롤 바를 요약, 선택 액션, 실행 액션 영역으로 재구성했습니다.
- 선택된 작업이 없을 때 상태 변경 및 담당자 적용 버튼을 비활성화해 오조작 가능성을 줄였습니다.
- 작업 row 선택 칩과 컨트롤 바 선택 버튼의 클릭 영역을 키웠습니다.
- 좁은 화면과 다크 모드에서 일괄 컨트롤 UI가 자연스럽게 보이도록 스타일을 보강했습니다.
- 확장 프로그램 버전을 `0.4.3`으로 올렸습니다.

### Fixed
- 작업 컨트롤 바의 그라데이션 배경이 바 위치와 어긋나 보이는 문제를 단색 배경으로 정리해 수정했습니다.

### Docs
- README의 일괄 처리 사용법을 새 버튼명과 선택 프리셋 흐름에 맞춰 갱신했습니다.
